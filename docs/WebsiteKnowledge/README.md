# EstateIQ Website Knowledge Pack

This folder translates the main EstateIQ application into website-building context for EstateIQ-Web.

EstateIQ is the product name. "The Financial Operating System for Small Landlords" is positioning.

## Purpose

Use this pack when building marketing pages, product pages, navigation, homepage sections, conversion copy, and trust messaging. It is written for future developers and AI coding assistants so website work stays grounded in the actual product.

## Source Basis

This pack was created by studying the main EstateIQ application and documentation, including:

- Root README and project architecture docs.
- Product, ICP, roadmap, ADR, architecture, security, onboarding, and financial workflow docs.
- Backend domains under `backend/apps/`.
- Frontend feature routes and components under `frontend/src/features/`.
- AI Copilot, billing ledger, expenses, document vault, document intake, property finance, tax documents, reports, subscriptions, and onboarding code/docs.

## Scope Rules

Do not make unsupported claims about production maturity, integrations, accuracy guarantees, automated tax compliance, banking connectivity, AI autonomy, or customer results.

Separate current product, planned product, and future roadmap in public copy whenever a feature is not already implemented.

Do not use fake testimonials, fabricated metrics, or generic SaaS language.

Do not describe EstateIQ as traditional property management software. It is an AI-native financial operating system for independent landlords and small real estate portfolios.
