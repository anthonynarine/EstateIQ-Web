# EstateIQ Brand Handbook

This folder is the source of truth for how EstateIQ should be described, designed, and assembled across the public website.

EstateIQ is the product name. "The Financial Operating System for Small Landlords" is positioning. Future website work should reinforce the EstateIQ brand consistently and avoid reintroducing any legacy placeholder name as a product name.

## Canonical Brand

Product name: EstateIQ

Primary tagline: The Financial Operating System for Small Landlords.

Short description: EstateIQ helps independent landlords understand what their portfolio earns, spends, and owes through deterministic financial records, document intelligence, and AI-powered insights.

Core positioning: EstateIQ is not traditional property management software. EstateIQ is an AI-native Financial Operating System built specifically for independent landlords and small real estate portfolios.
